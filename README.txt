Instruções para testar o sistema.
Para testar, você deverá ter instalado ou instalar na sua máquina o Python, em seguida baixe os arquivos do projeto. 

Abra o terminal (Prompt de comando) e instale o Flask.
pip install flask
 
Acesse a pasta com o arquivo extraído, por exemplo:
cd C:\caminho-da-sua-pasta
 
Execute o programa:
Python app.py
 

Abra seu navegador copie o endereço que aparecerá e acesse:
exemplo:
http://localhost:5000
